## ✨ Key Achievements & Implementations

### 1. Robust Node Abstraction (Clean Architecture)
Instead of duplicating code across multiple node files, I architected a highly reusable `<BaseNode />` component. 
* **DRY Principle:** Extracted common logic, wrappers, and styling into a single source of truth.
* **Scalability:** Creating a new node now takes just a few lines of code, passing only the unique inputs, outputs, and internal content.
* **Consistency:** Ensures that any global layout changes instantly reflect across all nodes in the pipeline.


### 2. Modern UI/UX & Styling
I transformed the unstyled interface into a modern, cohesive SaaS-like experience:
* **Design System:** Created a unified style for all inputs, selects, and textareas with consistent padding, rounded corners (12px), and clean borders.
* **Fluid Animations:** Integrated `framer-motion` to add professional micro-interactions. Nodes now feature a smooth pop-in animation on load and an elegant lift effect with a soft drop-shadow on hover.
* **Visual Hierarchy:** Implemented a color-coded top border for different node types (e.g., Green for Inputs, Red for Outputs, Blue for Text) to enhance visual scannability.
* **Floating Toolbar:** Redesigned the node creation toolbar into a modern, floating navbar layout.

## 🛠️ Tech Stack Used
* **React.js** (Hooks, State Management, Refs)
* **React Flow** (Node connections, Canvas handling)
* **Framer Motion** (UI micro-interactions and animations)
* **CSS / Inline Styling** (Custom scalable design system)
